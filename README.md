# Amethyst RP

Веб-сайт майнкрафт сервера

Написан на TypeScript с использованием Mantine UI компонентов

## Функционал

1. Подача заявки
2. Аунтефикация
3. Личный кабинет
4. Форум
5. Подача жалоб
6. Банковская система игровой валюты
7. Новости
